## Website Features â¡
- **Firebase Authentication**: Email/password + Google login, persistent sessions.
- **Role-Based Dashboard**: Separate user dashboard + **Admin Dashboard** (admin-only routes).
- **Premium Membership System**: Stripe checkout + premium activation via server webhook.
- **Lesson Management**: Create, update, delete lessons; set visibility (public/private) and access level (free/premium).
- **Favorites & Engagement**: Save lessons to favorites, view saved content, and track likes.
- **Reporting System**: Users can report lessons; admin can review/ignore/delete reported content.
- **Responsive Modern UI**: Clean dashboard layout, cards, tables, and charts for analytics.
